# 🌸 PROPOSAL WEBSITE FONT SETS 🌸

---

## SET 1 — BEST OVERALL (Recommended & ACTIVE)
**Vibe:** Cinematic + Premium + Soft Romance

### Fonts:
1. **Heading / Emotional Text:** Cormorant Garamond
   - Link: https://fonts.google.com/specimen/Cormorant+Garamond
   - Usage: Titles, basically everything emotional, proposal text, letter heading.
2. **Main UI / Paragraphs / Buttons:** Poppins
   - Link: https://fonts.google.com/specimen/Poppins
   - Usage: Buttons, letter body, captions, questions, small text.
3. **Signature Only:** Allura
   - Link: https://fonts.google.com/specimen/Allura
   - Usage: ONLY for "— Kushal 💞" signature.

**Vibe:** Elegant Luxury Romance

## SET 3 — Soft Korean Drama / Ghibli Vibe
**Vibe:** Dreamy Cozy Romance

### Fonts:
1. **Heading:** DM Serif Display
   - Link: https://fonts.google.com/specimen/DM+Serif+Display
   - Usage: Emotional, soft, artistic, storybook-like (Letter + Gallery).

---

### HOW TO SWITCH:
1. Copy the Google Font @import link into `App.css` or `index.html`.
2. Update the CSS variables in `:root` inside `App.css`.
